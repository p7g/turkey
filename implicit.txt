check_coherence(env, new_sig):
- unify the satisfied signature of the newly introduced canon against all existing ones. Any successful unification -> error

lookup(env, sig, args) -> module:
- invariant: type arguments to signature must be concrete
- walk the lexical scopes
  - unify the signature expression against those of all canon in the scope
  - on first match return that match
- if no match -> error
- if functor, instantiate each module param bound and then look it up, finally instantiate functor
